# 🌐 Red Empresarial para SurConnect IT Solutions
En este repositorio se van a manejar las entregas para la realización de una red empresarial para SurConnect IT Solutions, una empresa que ofrece servicios de outsourcing de TI con sede en Colombia pero para toda Sudamérica. Actualmente tiene 9000 empleados, pero se estima que en 5 años crezca hasta en un 30%.

En la zona de la Wiki del repositorio va a poder ver con mayor detalle todas las especificaciones técnicas y desrrollo del proyecto.
